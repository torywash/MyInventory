# MyInventory
A CLI Task manager script for keeping an inventory on things in your household. I created this to keep track of what was in my fridge and what books I had in my collection, but I will continue to use this highly customizable script for organization needs.

# How to Use
- Download the folder with the python files in it
- In an IDE of choice, run the script
- Choose between:
  - Fridge
  - Personal
  - Books
- Enjoy the script!
